Cheat Sheet — Chapters 1–6

Chapter 1 — Program Skeleton & I/O

#include <iostream>
using namespace std;
int main() {
    cout << "Hello\n";   // newline: \n or endl
    return 0;
}
  • // single‑line, /* ... */ multi‑line comments.
  • cout (output), cin (input). Use prompts so users know what to enter.

Chapter 2 — Variables & Types

int count = 0;        // whole numbers
double rate = 19.75;  // floating point
char letter = 'A';    // single char
bool ok = true;       // true/false
string name = "Marc"; // text
  • Choose types that match data (e.g., money → double, counts → int).
  • sizeof(type) gives bytes (platform/compiler dependent).

Chapter 3 — Assignment, Input, Formatting

double total = price * qty;
cout.setf(ios::fixed); cout.setf(ios::showpoint);
cout.precision(2); // money
  • Order of evaluation matters; integer division truncates.
  • Always validate user input (see do‑while sample below).

Chapter 5 — Loops

for (int i=0; i<5; ++i) cout<>n; sum+=n; }    // sentinel
int x; do { cin>>x; } while(x<1||x>10); // validate
  • Pick for when count is known; while for sentinel; do‑while when you need to run at least once.

Chapter 6 — Functions

double area(double L,double W); // prototype
int main(){ cout << area(3,4); }
double area(double L,double W){ return L*W; }
  • Return exactly one value. Parameters here are passed by value.
  • Put prototypes before main or define functions before use.

Loop Pattern Cookbook extra practice

1) Multiplication table (1..N)

int N=5;
for (int r=1; r<=N; ++r){
  for (int c=1; c<=N; ++c){
    cout << (r*c) << "\t";
  }
  cout << "\n";
}

2) Sum of digits

int n=7531, sum=0;
while(n>0){ sum += n%10; n/=10; }

3) Factorial (n!)

int n=5; long long f=1;
for(int i=2;i<=n;++i) f*=i; // 120

4) GCD (Euclid)

int a=48,b=18;
while(b!=0){ int r=a%b; a=b; b=r; }
cout << a; // 6

5) Triangle pattern

int rows=4;
for(int r=1;r<=rows;++r){
  for(int c=1;c<=r;++c) cout<<"*";
  cout<<"\n";
}

6) Prime test (basic)

int n=29; bool prime=true;
for(int d=2; d*d<=n; ++d){
  if(n%d==0){ prime=false; break; }
}
cout<<(prime?"prime":"not");

Final Project Walkthrough — Payroll (Ch. 1–6 only)

Goal: compute pay for one or more employees using input, selection, loops, and functions (no arrays or files to stay within Ch. 1–6). Optional upgrade for file I/O is shown at the end (ignore if your course limits to 6 chapters).

Requirements (example)

  • Input per employee: name (string), hours (double), rate (double).
  • Overtime: hours over 40 paid at 1.5× rate.
  • Taxes: use constants (e.g., FED=0.12, STATE=0.05).
  • Compute: gross, tax, net. Format money to 2 decimals.
  • Allow multiple employees until user types done as the name (sentinel), and show totals.
  • Validate: hours ≥ 0 (and maybe ≤ 80), rate ≥ 0.

Pseudocode

loop:
  ask for name
  if name == "done" stop
  read hours, rate (validate)
  gross = computeGross(hours, rate)
  tax   = computeTax(gross)
  net   = gross - tax
  printPayStub(...)
  update running totals
end loop
print overall totals

Functions

double computeGross(double h, double r){
    const double OT = 40.0, OT_MULT = 1.5;
    if (h <= OT) return h * r;
    return OT * r + (h - OT) * r * OT_MULT;
}

double computeTax(double gross){
    const double FED=0.12, STATE=0.05;
    return gross * (FED + STATE);
}

void printMoney(double v){
    cout.setf(ios::fixed); cout.setf(ios::showpoint); cout.precision(2);
    cout << v;
}

void printPayStub(const string& name, double h, double r, double g, double tax, double net){
    cout << "----- PAY STUB -----\n";
    cout << "Name: " << name << "\n";
    cout << "Hours: " << h << "  Rate: $"; printMoney(r); cout << "\n";
    cout << "Gross: $"; printMoney(g); cout << "  Tax: $"; printMoney(tax);
    cout << "  Net: $"; printMoney(net); cout << "\n\n";
}

Main (sentinel loop • no arrays • no files)

#include 
#include 
using namespace std;

double computeGross(double h, double r);
double computeTax(double gross);
void   printMoney(double v);
void   printPayStub(const string& name, double h, double r, double g, double tax, double net);

int main(){
    string name;
    double hours, rate;
    double totalGross=0, totalTax=0, totalNet=0;
    int count=0;

    while (true){
        cout << "Employee name (or 'done'): ";
        if (!(cin >> name)) return 0; // input failure
        if (name == "done") break;

        // input validation (do-while)
        do {
            cout << "Hours worked: ";
            cin  >> hours;
        } while (hours < 0 || hours > 80);

        do {
            cout << "Hourly rate: ";
            cin  >> rate;
        } while (rate < 0);

        double gross = computeGross(hours, rate);
        double tax   = computeTax(gross);
        double net   = gross - tax;

        printPayStub(name, hours, rate, gross, tax, net);

        totalGross += gross; totalTax += tax; totalNet += net; ++count;
    }

    cout.setf(ios::fixed); cout.setf(ios::showpoint); cout.precision(2);
    cout << "----- TOTALS (" << count << " employees) -----\n";
    cout << "Gross: $" << totalGross << "  Tax: $" << totalTax
         << "  Net: $" << totalNet << "\n";
    return 0;
}
